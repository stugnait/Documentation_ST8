# 📄 System Specification Document (SSD)

## Основні функції (Functional Requirements)
- **FR-01: Генерація QR**. Створення криптографічно захищеного коду.
- **FR-02: Валідація**. Перевірка коду через Telegram-бота.
- **FR-03: Anti-Fraud**. Захист від повторного проходу (double-spend).
- **FR-04: Аналітика**. Оновлення лічильників на дашборді в реальному часі.

## Нефункціональні вимоги
| Категорія | Вимога | KPI |
| :--- | :--- | :--- |
| **Продуктивність** | Швидкість бота | < 300 мс на скан |
| **Доступність** | Аптайм | 99.9% під час івенту |
| **Безпека** | Авторизація | JWT-токени |

> **Обмеження:** Система працює тільки онлайн. Офлайн-режим не передбачений архітектурою.
