# ✅ Definition of Done (Documentation)

Жодна задача не вважається закритою, поки не виконані ці вимоги:

| Тип зміни | Обов'язкова дія | Де перевірити? |
| :--- | :--- | :--- |
| **Новий ендпоінт** | Додати опис у `openapi.yaml` | Swagger UI |
| **Зміна логіки** | Оновити SSD/SDD файли | MkDocs Site |
| **Нова кнопка** | Створити Story в Storybook | UI Library |
| **Баг-фікс** | Оновити тест-кейс (якщо треба) | Traceability Matrix |

> **Важливо:** Якщо змінився контракт API (Breaking Change), версія документації має бути піднята (наприклад, з v1.0.0 до v1.1.0).
