# 🚀 Швидкий старт для розробника

Радий бачити тебе в команді! Ось короткий гайд, як розгорнути проект на "локалці" і не збожеволіти.

## Що треба встановити?
- **Docker & Docker Compose** — без цього навіть не починай.
- **Python 3.10+** (або Go, залежно від твого мікросервісу).
- **Ngrok** — щоб Telegram міг надсилати вебхуки на твій комп.

## Як запустити проект?
1. **Клонуй реп:** `git clone [url_проекту]`
2. **Налаштуй оточення:** Скопіюй `.env.example` у `.env` і пропиши там свій `TELEGRAM_BOT_TOKEN`.
3. **Підніми інфраструктуру:**
   ```bash
   docker-compose up -d
